Это мой проект в рамках решения ИИ-задачника. Здесь я попробовал реализовать сложную структуру роутинга и кастомный дизайн при помощи LLM-моделей 👾.

## Что я сделал:
- Настроил многоуровневую структуру папок: `/i/am/designer`.
- Использовал **Layouts** для единого оформления страниц.
- Стилизовал элементы с помощью **Tailwind CSS**.
- Настроил CI/CD через Vercel.
