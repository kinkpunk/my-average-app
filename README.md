### 👾 О проекте
Это мой первый шаг в разработку на **Next.js**, выполненный в рамках ИИ-задачника. В этом проекте я исследовал возможности совместной разработки с LLM-моделями, уделяя внимание архитектуре и чистоте кода.

[Посмотреть живое демо](https://vercel.app) ← *замени на свою ссылку*

---

### 🛠 Что реализовано:
- [x] **Сложный роутинг:** Настроена многоуровневая структура вложенных путей `/i/am/designer`.
- [x] **Архитектура Layouts:** Применено единое оформление для целых разделов сайта через файлы макетов.
- [x] **Современный UI:** Адаптивная стилизация и темная тема с помощью **Tailwind CSS**.
- [x] **Автоматизация (CI/CD):** Настроен мгновенный деплой через **Vercel** при каждом обновлении кода.

### 🧪 Технологический стек:
* **Framework:** Next.js 15+ (App Router)
* **Styling:** Tailwind CSS
* **Deployment:** Vercel / GitHub Actions
* **AI Tooling:** Совместная работа с LLM
